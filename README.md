# CS2340-Server
This is the server that we are going to use for CS2340

**Requirements**  
`sudo apt-get install python-minimal
sudo apt-get install python-mysqldb`


**MySQL installation**  
`sudo apt-get update
sudo apt-get install mysql-server`

Login to MySQL and then type the password  
`mysql -u root -p`
